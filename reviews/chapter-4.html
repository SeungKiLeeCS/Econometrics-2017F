Chapter 4 : Multiple Linear Regression

Multiple Linear Regression

  • MLR.6 : Normality of the Error Term

This assumes unobserved factors and their variance are normally distributed around the population regression function regardless of explanatory variable.

, Independent from

The error term is the sum of many different unobserved factors.

Sum of independent factors are normaly distributed by Central Limit Theorem.

  • Normality and its problems

Problems:

  1. How many is "many" factors? What number do we need to have? -> Sample Size 31, Degree of Freedom 120
  2. Possibly very heterogenuous distribution of individual factors -> Assumption is wrong for certain samples.
  3. How independent are the different factors? -> Colinearity Problem

Really the biggest problem with normality assumption of error terms is that it is empirical. In many cases normality is impossible or questionable by definition. However Inductively we can conclude that at the very least the error distribution will be "close" to normal distribution.

Normality does not hold for values that cannot be negative like wage or time, and values that will inevitably have limitted integer outcomes such as number of arrests (most people would have arrests ranging from 0 ~ 10.), or dummy variables such as female, or unemployeed (binary value.)

As mentioned in Chapter 3, the best way to normalize the data is by log transformation. First try taking log of dependent variable.

Normality is the most important assumption in using OLS because this trait determines the BUE (doesn't even have to be linear!) of estimator.

For Statistical Inference, normality can be substituted by large sample size.

While MLR.1 - MLR.5 is referred to as the Gauss-Markov Assumption, including normality, MLR.1 - MLR.6 is referred to as "Classical Linear Model(CLM) assumptions"

  • Unrestricted Model (Do not drop insignificant variables)

Without even doing the full t-test, you can see that all independent variables past x2 will be insignificant at 95% confidence interval because the standard deviation is bigger than the coefficient, or only slighly smaller. We know that this will force t-statistic to be smaller than 1.00 for most of these, and we already saw that even with alpha value as large as 0.2 the critical value is 1.282, which is still greater than 1.00.

So for above model, we will run a F-test.

Note that we have SSR this time. If the insignificant variables are dropped, the SSR will increase and goodness of fit will decrease by definition. What we want to know is whether this increase in SSR is significant. In other words, does dropping seemingly insignificant variables (that we found using t-test) change anything in regression?

  4. Rejection Rule

F-distribution is always positive in value as evident by the fact that SSR can only increase if variables are dropped. Because of this, there is no two tail test like t-test. Rather this one form of F-testing works for general multiple linear hypotheses.

  • Why do we need this test?

As you can see from our example, three variables that were insignificant in individual t-test turned out to be significant in joint f-testing. This sugguests that these three variables have multicolinearity. A good way to get rid of the structural multicolinearity would be actually doing more research and getting more data so you can come up with one composite variable that will replace all three variables. In the mean time, we use F-test as the bandage to run the regression while someone else fixes the problem with dataset.
